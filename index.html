Datos diarios de temperatura mínima y máxima.

Datos diarios de temperatura mínima y temperatura máxima pueden ser consultados en este blog. Las tablas son actualizadas una vez por año. Es mi intención mudar estos datos a esta página, ya que el proceso de actualizarlas en forma mensual resulta sustancialmente menos tedioso, sin embargo me temo que el espacio que ocuparán los archivos html asociados con estos datos será excesivo.

Anomalías ponderadas de temperatura al norte de los 40°S.

A partir del año 1971, para cada mes, se calculó una anomalía de la temperatura mínima media y una temperatura máxima media mediante una metodología descripta en este posteo.
