Cálculo de la temperatura media mensual.
La temperatura media mensual (TMM) es el resultado de promediar todos los datos de temperatura observados durante el mes en las horas principales 3, 9, 15 y 21 HOA (6, 12, 18 y 00 UTC, también conocidas como 6z, 12z, 18z y 0z).
Por diferentes razones, puede ocurrir que de forma temporal las observaciones se vean limitadas a horarios diurnos, o a horarios que no incluyan a la observación de las 6z, lo que ocasiona que no se pueda calcular una TMM. Para estos casos, se recurre a la elaboración de un factor de corrección, que en términos generales consiste en la diferencia promedio entre la TMM calculada con y sin la observación de las 6z. Así, cada vez que falten las observaciones de las 6z y esto impida el cálculo de una TMM, se promedian las horarias restantes, y a este promedio se le suma el factor de corrección que corresponda. Así es como de manera oficial se calcula una TMM en Argentina (ver mensajes CLIMAT publicados en Ogimet). En otros países el procedimiento es diferente.
El algoritmo usado para calcular las TMM fue el siguiente:

  1. Seleccionar los datos horarios correspondientes a las horas principales.
  2. Agrupar los datos por mes y año.
  3. Para cada grupo, calcular el porcentaje de datos faltantes para cada hora principal.
  4. Si el porcentaje mencionado en 3 no supera el 10% para ninguna hora principal, promediar todas las observaciones dentro del grupo. Esta será su TMM.
  5. Si el porcentaje mencionado en 3 supera el 10% para la hora principal 9, 15 o 21, la TMM será igual a NA.
  6. Si el porcentaje mencionado en 3 supera el 10% para la hora principal 3, pero no para las horas principales 9, 15 o 21, promediar estas últimas observaciones y sumarle el factor de corrección que le corresponda. Esta será su TMM.

Cálculo de la temperatura media estacional y la temperatura media anual.
En el hemisferio sur, el verano climatológico comprende los meses de diciembre, enero y febrero (DEF), el otoño climatológico los meses de marzo, abril y mayo (MAM), el invierno climatológico los meses de junio, julio y agosto (JJA), y la primavera climatológica los meses de septiembre, octubre y noviembre (SON). La temperatura media estacional (TME) es el promedio de las TMM que conforman la estación climatológica en cuestión. Como ya se mencionó, el verano comprende el mes de diciembre del año A y los meses de enero y febrero del año siguiente, A+1, por lo que su TME aparecerá computada para este último año.
Por otra parte, la temperatura media anual (TMA) es el promedio de las 12 TMM que comprenden el año. Es requisito tanto para la TME como para la TMA que no falte el dato de ningún mes.

Factores de corrección
Cada mes cuenta con un factor de corrección. El algoritmo para determinarlo es el siguiente:

  1. Seleccionar datos horarios previos al año 2021.
  2. Seleccionar datos horarios correspondientes a horas principales (3,9,15,21 HOA).
  3. Para cada día, calcular una temperatura media con los datos de las horas 9, 15 y 21. Si hay faltantes, el resultado es NA.
  4. Para cada día, calcular una temperatura media con los datos de las horas 3, 9, 15 y 21. Si hay faltantes, el resultado es NA.
  5. Si el resultado de hacer 3 es NA, remover el día. Hacer lo mismo si el resultado de hacer 4 es NA.
  6. Para cada día, hacer la diferencia entre la temperatura media calculada en 4 y la temperatura media calculada en 3.
  7. Para cada mes, seleccionar las últimas 300 diferencias, o la cantidad máxima disponible.
  8. Para cada mes, realizar el promedio de las diferencias consideradas en el punto anterior. Este promedio es el factor de corrección.

Casos especiales.
Puerto Deseado: Tiene dos problemas. El primero es que no tiene la cantidad de observaciones 6z suficientes para que tenga factores de corrección, por lo que se usaron los factores de corrección de San Julián. El segundo es que entre 1979 y 2011 no hubo observaciones los dias domingos, y esto hace que los dias de datos faltantes excedan el mínimo para calcular temperaturas medias mensuales. Se decidió rellenar estos faltantes con datos de San Julián, mediante modelos de regresión lineal mensuales.
