Mejorando los factores de corrección: Un caso de estudio con datos de Bahía Blanca.

¿Qué tan buenas son las estimaciones?

Usaremos datos de Bahía Blanca Aero, desde enero de 1956 hasta el presente (septiembre 2025). Para cada mes y año se calcula la TMMsin6z, que es la temperatura media mensual sin el dato de las 6z, es decir utilizando todas las horas principales excepto la de las 6z. Luego, se le resta el factor de correción que corresponda (que es según el mes). Lo que obtenemos con esta diferencia es una temperatura media mensual estimada, que llamaremos TMM_est. Luego, compararemos cada TMM_est con la temperatura media real, TMM, haciendo la diferencia entre las dos.
