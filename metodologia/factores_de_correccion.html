Factores de corrección
Cada mes cuenta con un factor de corrección. El algoritmo para determinarlo es el siguiente:
  1. Seleccionar datos horarios previos al año 2021.
  2. Seleccionar datos horarios correspondientes a horas principales (3,9,15,21 HOA).
  3. Para cada día, calcular una temperatura media con los datos de las horas 9, 15 y 21. Si hay faltantes, el resultado es NA.
  4. Para cada día, calcular una temperatura media con los datos de las horas 3, 9, 15 y 21. Si hay faltantes, el resultado es NA.
  5. Si el resultado de hacer 3 es NA, remover el día. Hacer lo mismo si el resultado de hacer 4 es NA.
  6. Para cada día, hacer la diferencia entre la temperatura media calculada en 3 y la temperatura media calculada en 4.
  7. Para cada mes, seleccionar las últimas 300 diferencias, o la cantidad máxima disponible.
  8. Para cada mes, realizar el promedio de las diferencias consideradas en el punto anterior. Este promedio es el factor de corrección.
